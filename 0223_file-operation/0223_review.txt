## 身についたこと
- pathlibによるディレクトリ作成: mkdir(parents=True, exist_ok=True) を使うことで、エラーを防ぎながら階層構造のディレクトリを自動作成できるようになった。
- ファイルの移動とコピーの使い分け: pathlib.Path.rename() による移動（名前変更）と、shutil.copy() による複製の実装方法を習得した。
- パスの結合とオブジェクト操作: 文字列を / 演算子で繋ぐ Path オブジェクト特有の書き方を学び、OSに依存しない堅牢なコードの書き方を理解した。

## 実務で役にたつ想定場面
- バックアップの自動生成: 毎日更新される業務日報やログファイルを、別フォルダへ日付付きの別名でコピーして保存するスクリプト。
- ファイル整理の自動化: ITサポートデスクで受け取った大量の添付ファイルを、拡張子や特定のキーワードに基づいて適切なフォルダ（backup や archive など）へ自動で振り分ける。

- 初期設定スクリプトの作成: 新しいプロジェクトを始める際に、必要なディレクトリ構造をコマンド一つで一括作成するツールの構築。

## error
```
FileNotFoundError: [Errno 2] No such file or directory: 'old_file.txt' -> 'backup/new_file.txt'
```
原因:

- 移動先のディレクトリ（backup）が存在しなかった。
- プログラムを実行しているカレントディレクトリが、対象のファイルがある場所とズレていた。

解決策:
- Path.mkdir() を使用して、移動前にディレクトリが存在することを保証するようにコードを修正した。
- ターミナルで cd コマンドを使い、正しい作業ディレクトリに移動してから実行した。
